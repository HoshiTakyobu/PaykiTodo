## v1.6.0

- 新增手机端局域网网页控制台，实现 PaykiTodo 电脑同步第一版
- 设置页新增“电脑同步”入口，可开启服务、查看局域网地址、查看和重置访问密钥

- 新增同步前台服务，支持应用重启、设备重启后的自动恢复
- 内置 Wiki 与 README 同步补充电脑同步的完整使用说明
- 版本号升级到 `1.6.0` / `versionCode 72`
