# Code of Conduct

PaykiTodo is maintained as a practical personal productivity tool. Project discussion should stay focused, respectful, and useful.

## Expected behavior

- Be clear and concrete when reporting problems.
- Criticize code and product behavior, not people.
- Remove private data from screenshots, logs and examples.
- Respect that maintainer time is limited.

## Unacceptable behavior

- Harassment, insults, threats or discriminatory language.
- Posting another person's private data.
- Posting API keys, tokens, signing material or private endpoints.
- Repeated low-effort comments that do not help reproduce or solve an issue.

## Enforcement

The maintainer may edit, hide, lock or delete discussions and issues that violate these rules. Severe or repeated violations may lead to blocking from the repository.
