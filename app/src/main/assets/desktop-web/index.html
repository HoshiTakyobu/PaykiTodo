规划台使用说明

先像备忘录一样写；识别后在预览里确认，才进入正式待办和日程。

1. 直接写

规划台就是上游备忘录。先把脑子里的任务、课程、DDL、会议和想办的事写下来，不需要先决定它属于待办还是日程。

10:00-12:00 写论文 @图书馆3楼
12:00-13:00 吃饭
任务M ddl 15:00
明天 16:30 交材料
整理保研材料
打印成绩单

2. 常见自然写法

  • 日程：10:00-12:00 写论文 @图书馆3楼
  • 逗号日程：10:00-12:00, 【课程】习思想, "@主楼B1-412"
  • DDL：交材料 ddl 5.29 23:59、任务M ddl 15:00、明天 16:30 交材料
  • 无 DDL 待办：整理保研材料。导入后会作为今日待办显示。
  • 地点：可以写 @图书馆3楼、地点：主楼B1-412，也可以用引号包起来。

3. 识别和导入

  1. 写完后点“识别计划”。如果设置里启用了 AI 源，会优先走 AI；失败或未配置时回到本地规则。
  2. 右侧会出现待办、日程、跳过和错误条目。识别结果只是预览，不会直接入库。
  3. 在预览里修正标题、DDL、开始/结束时间、地点、分组、提醒和备注。
  4. 勾选需要导入的条目，点击“导入选中”。至少勾选一条有效候选后才能导入。
  5. 导入后不会往你的原文里追加井号标签；已导入状态由软件内部记录。

4. 写得乱也没关系

规划台的目标不是逼你记语法，而是把“先粗略写下来”变成正式待办/日程之前的缓冲区。看不准的地方就在预览里改。
