# Support

PaykiTodo is a personal open-source Android project. Support is best-effort.

## Before asking for help

Please check:

1. The latest GitHub Release.
2. `README.md` for installation and build notes.
3. `CHANGELOG.md` for recent behavior changes.
4. In-app Wiki / help pages when the question is about usage.

## Where to ask

- Bug reports: open a GitHub Issue using the bug report template.
- Feature requests: open a GitHub Issue using the feature request template.
- Security-sensitive reports: follow `SECURITY.md` and avoid posting secrets publicly.

## What to include

For bug reports, include:

- app version, such as `1.13.11`;
- Android version and device model;
- whether the build is debug or release;
- clear reproduction steps;
- expected behavior and actual behavior;
- screenshots only after removing private schedule data.

Do not include API keys, tokens, signing files, private Base URLs, backups, or personal logs in public issues.
